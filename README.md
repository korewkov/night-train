# Путь проводника

Интерактивная визуальная новелла для обучения студентов-проводников.

## Ссылка на сайт

https://korewkov.github.io/night-train/

## Что есть сейчас

- главная страница
- выбор истории
- интерактивный сценарий «Ночной рейс»
- моральные выборы
- несколько финалов
- служебные замечания
- сохранение прогресса в браузере через localStorage
